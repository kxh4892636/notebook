# react 基础

## 创建 react 项目

**环境依赖**

- nodejs;

**创建 react 项目**

```
npx create-react-app your-app-name
```

## 工具链

- 包管理器;
  - npm: nodejs 自带.
- 编辑器;
  - vscode;
    - 插件;
      - ESLint;
      - Prettier;
      - ES7+ React/Redux/React-Native snippets;
- 编译器;
  - ts: 类型提示;
- 打包工具;
  - webpack: 最流行;
- 代码格式检测;
  - ESLint;
- 代码测试;
  - Jest;
- 工具包;
  - immer: 不可变类型;
- 浏览器插件;
  - React Developer Tools;
