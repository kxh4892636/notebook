# transform

## transform 属性

**作用**

- 对标签进行形状变换.

**语法格式**

```css
div {
  transform: translate(30px, 20px) rotate(20deg);
}
```

**多值语法**

- 随意写,
- 空格分隔.

### 函数

#### matrix 函数

**作用**

- 对标签进行矩阵变换.

#### perspective 函数

**作用**

- 设置对于标签的透视距离.

#### rotate 函数

**作用**

- 旋转标签.

#### translate 函数

**作用**

- 平移标签.

#### scale 函数

**作用**

- 比例放缩.

#### skew 函数

**作用**

- 倾斜标签.
