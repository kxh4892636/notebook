# wsl 教程

## 概述

### 基本命令

```bash
# 安装 wsl
wsl --install
# 安装指定版本 wsl
wsl --install -d <Distribution Name>
# 列出可用的 linux 版本
wsl --list --online
# 列出已安装的 linux 发行版
wsl --list --verbose
# 更新 wsl
wsl --update
# 检查 wsl 状态
wsl --status
# 关闭 wsl
wsl --shutdown
```

## 教程

### 最佳安装实践

**安装**

```bash
wsl --install
```
