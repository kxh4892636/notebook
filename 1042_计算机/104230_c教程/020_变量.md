# 变量

## 变量名

**规则**

- 字母, 下划线开头;
- 不能以数字开头;
- 不能超过 63 个字符;
- 不能使用保留字和关键字;

## 声明

```c
int height;
```

## 赋值

```c
// 先声明再赋值
int num;
num = 42;

// 同时声明并赋值
int num = 42;
```

## 作用域

**文件作用域**

- 声明在文件顶层的;
- 作用域整个文件;

```c
int x = 1;

int main(void) {
  printf("%i\n", x);
}
```

**块作用域**

- 声明在 {} 内部;
- 在 {} 有效;

```c
int a = 12;

if (a == 12) {
  int b = 99;
  printf("%d %d\n", a, b);  // 12 99
}

printf("%d\n", a);  // 12
printf("%d\n", b);  // 出错
```

**作用域的优先级**

- 若多个作用域具有相同变量;
- 优先使用内层作用域的变量;

```c
{
  int i = 10;
  {
    int i = 20;
    printf("%d\n", i);  // 20
  }
  printf("%d\n", i);  // 10
}
```
